User manual for lecturers

Schedule

How to add an event to the schedule?
  1. Go to the schedule page by clicking the «CATS» logo in the upper left corner of the screen.
  2. Double-click a cell that meets the time you need.
  3. You will see a pop-up window. Select the required event type there. It can be either a lesson or an event.
  4. Enter the necessary data.
  5. Press the [Save] button.
How to edit an event?
  1. Find the event you want to edit and press the [...] button.
  2. There will be Edit and Delete options.
  3. Press the [Edit] button.
  4. You will see a pop-up window. Edit the data you need there.
  5. Press the [Save] button.
How to delete an event?
  1. Find the event you want to delete and press [...] button.
  2. There will be Edit and Delete options.
  3. Press the [Delete] button.
  4. You will see a pop-up window. Click [Yes] to confirm deletion or [No] to cancel it.

Courses

How to add a course?
  1. Click the [Courses] button in the top menu.
  2. Press the [Course management] button on the following page.
  3. Then click the [Add course] button.
  4. Fill in the data. Please note that the News unit is included automatic, the other ones are optional. You are able to edit this information whenever you want.
  5. Press the [Save] button.
  6. The added course will appear in the courses list.
How to edit a course?
  1. Click the [Courses] button in the top menu.
  2. Press the [Course management] button on the following page.
  3. Then click the [Edit course] icon.
  4. Edit the data you need and press the [Save] button.
  5. The edited course will appear in the courses list.
How to delete a course?
  1. Click the [Courses] button in the top menu.
  2. Press the [Course management] button on the following page.
  3. Then click the [Delete course] icon.
  4. You will see a dialog window. If you are sure you want to delete the course, press the [Delete] button, otherwise press [Cancel].
  5. The course will be deleted and you will no longer see it in the courses list.
How to join a lector?
  1. Click the [Courses] button in the top menu.
  2. Press the [Course management] button on the following page
  3. Find the required course in the list and click the [Join lector] icon.
  4. There will be a dialog window. Select the lector you need and press the [Join] button.
  5. The lector you chose is now joined and can manage it.
How to enter a course page?
  1. Press the [Courses] button in the top menu.
  2. Click the [Choose course] label on the following page.
  3. There will be a drop-down menu. Select the course you need there.
  4. You will be redirected to the page of the required course.

News

How to add news?
  1. Enter a course page.
  2. Go to the News section.
  3. Press the [Add news] button.
  4. There will be a pop-up window, where you are to input data. Then press the [Save] button.
  5. The news you added will be shown in the list.
How to edit news?
  1. Enter a course page.
  2. Go to the News section.
  3. Hover the mouse over required news. Click the [Edit] icon.
  4. There will be a pop-up window, where you are to edit data. Then press the [Save] button.
  5. The changes you made will be shown in the news list.
How to delete news?
  1. Enter a course page.
  2. Go to the News section.
  3. Hover the mouse over required news. Click the [Delete] icon.
  4. There will be a pop-up window. If you are sure you want to delete the news, press the [Delete] button, otherwise press [Cancel].
  5. The news will be deleted and you will no longer see in the news list.
How to hide news on the homepage?
  1. Enter a course page.
  2. Go to the News section.
  3. Press the [Hide all news] button.
  4. All the news will be hidden and will not be shown on the homepage. Hidden news will be displayed on the left of the news section. The color of the news will be switched to gray.
How to show news on the homepage?
  1. Enter a course page.
  2. Go to the News section.
  3. Press the [Show all news] button.
  4. All the news will be no longer hidden and will be shown on the homepage. Every news will have its own color you assigned to it.

Lectures

How to add a lecture?
  1. Enter a course page.
  2. Go to the Lectures section.
  3. Press the [Add lecture] button.
  4. Input data in a pop-up window. Press the [Save] button.
  5. The lecture will be shown in the lectures list.
How to add or delete lecture's date?
  1. Enter a course page.
  2. Go to the Lectures section.
  3. Then go to the Attending lectures tab.
  4. Press the [Schedule management] button.
  5. Enter the required data in a pop-up window and press the [Save] button.
  6. The date you added will be shown in the list below. If you want to delete it, press the [Delete] icon in this list.
How to delete dates of all the lectures?
  1. Enter a course page.
  2. Go to the Lectures section.
  3. Then go to the Attending lectures tab.
  4. Press the [Delete all dates] button.
  5. You will see a pop-up window. Press [Delete] to confirm deletion. Otherwise press [Cancel].
  6. The dates of all lectures will be deleted.
How to mark lectures attendance?
  1. Enter a course page.
  2. Go to the Lectures section.
  3. Then go to the Attending lectures tab.
  4. Double-click any cell that matches the required date.
  5. You are able to mark every absent student in a pop-up window by entering the required information. When you finish press the [Save] button.

How to check and mark students’ laboratory works?

  3. Then ho to the Works submission tab.

Settings

How to confirm or block access to Educats for a student?
  1. In the top menu there is a [New students] icon. Click it.
  2. You will be redirected to a new page. You are able to find the student you need by entering their group or name.
  3. Look for the student you need and click the [Confirm access] or [Block access] icon.
How to edit personal account data?
  1. Click the [Personal account] icon in the top menu.
  2. You will be redirected to a new page. You are able to upload an image or edit information about yourself there.
  3. Press the [Save changes] button if you want to confirm changes. Otherwise press [Close].
How to change password?
  1. Click the [Personal account] icon in the top menu.
  2. Look for the [Change password] button on the next page and press it.
  3. You will see a dialog window where you are to enter the new password. Press the [Change] button if you want to confirm changes. Otherwise press [Cancel].

Tests

How to add a test?
  1. Enter the course page.
  2. Go to the Tests section.
  3. Press the [Add test] button.
  4. In the pop-up page fill in the fields and press the [Save] button.
How to edit a test?
  1. Enter the course page.
  2. Go to the Tests section.
  3. Find the desired test and click the [Edit test] icon.
  4. In the pop-up page edit the required fields and press the [Save] button.
How to delete a test?
  1. Enter the course page.
  2. Go to the Tests section.
  3. Find the desired test and click the [Delete test] icon.
  4. In the pop-up page press the [Yes] button.
How to add questions to the test?
  1. Enter the course page.
  2. Go to the Tests section.
  3. Find the desired test and click the [Go to questions] icon.
  4. In the pop-up page press the [Add question] button (to add a new question) or press the [Add a question from another test] button (to add a question from an existing test).
  5. Fill in the fields or select a question from an existing test and press the [Save] button.
How to edit a question in a test?
  1. Enter the course page.
  2. Go to the Tests section.
  3. Find the desired test and click the [Go to questions] icon.
  4. In the pop-up page find the necessary question and click the [Edit question] icon.
  5. Edit the required fields and press the [Save] button.
How to delete a question in a test?
  1. Enter the course page.
  2. Go to the Tests section.
  3. Find the desired test and click the [Go to questions] icon.
  4. In the pop-up page find the necessary question and click the [Delete question] icon.
  5. In the pop-up window press the [Yes] button.
How to open a test for students to pass?
  1. Enter the course page.
  2. Go to the Tests section.
  3. Find the desired test and click the [Test availability] icon.
  4. In the pop-up page select group, press the [Open for everyone] button (to open the test for all the students in a selected group). If the test needs to be opened for a particular student, then click the [Test availability] icon beside him (blue color icon - test is closed, white - test is opened).
  5. Configure test availability and press the [Close] button to close the window.
How to close a test for students to pass?
  1. Enter the course page.
  2. Go to the Tests section.
  3. Find the desired test and click the [Test availability] icon.
  4. In the pop-up page select group, press the [Close for all] button (to close the test for all the students in a selected group). If the test needs to be closed for a particular student, then click the [Test availability] icon beside him (blue color icon - test is closed, white - test is opened).
  5. Configure test availability and press the [Close] button to close the window.
How to pass a test to verify it?
  1. Enter the course page.
  2. Go to the Tests section.
  3. Find the desired test and click the [Pass the test] icon.
  4. In the pop-up page press the [Next] button.
  5. The test will be opened for passing. Press the [Reply] button to save current answer or press the [Skip] button to not save the current answer and return to the question in the end.
How to view test results?
  1. Enter the course page.
  2. Go to the Tests section.
  3. Go to the Test results tab.
  4. Select the group, tests, students.
  5. The results will be displayed for every student in every test.
How to monitor the passing of the test?
  1. Enter the course page.
  2. Go to the Tests section.
  3. Enter the Test passing monitoring tab.
  4. The names of students who are currently passing the test will be displayed.

Graduation project

Who has access to Graduation project section?

Graduation project section can be access only by 3 roles:

- Teacher (head of GP, this is configured by the administrator);

- Student (his graduation year matches current year);

- SEC secretary (teacher, may not be head of GP, configured by the administrator).
